## 1.0.0

* Initial release
* Unified notification API: local + push + scheduled notifications in 3 lines
* Built-in cron parser for recurring notifications
* Automatic timezone handling — no TZDateTime required
* FCM integration (optional, runtime-detected)
* Notification channels with auto-creation
* Auto-grouping (InboxStyle on Android, threads on iOS)
* Action buttons with inline reply support
* Notification history with query API and unread count
* Analytics callbacks for delivery, open, dismiss, and action events
* Pre-built widgets: NotifyBanner, NotifyInbox, NotifyBadge
* bg_orchestrator integration via BgTaskTrigger
* Example app with 7 demo screens
